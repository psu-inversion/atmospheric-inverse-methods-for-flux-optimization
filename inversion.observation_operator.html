inversion.observation_operator module¶

Utilities for working with influence function files.

Designed for files dimensioned: observation_time, site, forecast_period, y, x.

Functions here turn these into BSR matrices dimensioned observation, time, y, x or observation_time, site, time, y, x keeping the memory savings of the first form while still usable for standard code.

I should mention at some point that “influence function” and “observation operator” refer to the same quantity, represented by \(H\) in the equations. This is used to represent which fluxes influence each observation, as well as what observations would be expected given a specific set of fluxes.
