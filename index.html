Welcome to Inversion’s documentation!¶

This package contains functions and classes developed for the application of inverse methods to the optimization of carbon dioxide surface fluxes using atmospheric mixing ratio measurements. The recommended function is inversion.optimal_interpolation.save_sum(), which is best able to take advantage of functionality offered by inversion.linalg.DaskKroneckerProductOperator().
