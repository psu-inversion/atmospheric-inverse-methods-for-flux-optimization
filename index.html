Atmospheric Inverse Methods for Flux Optimization¶

This package contains functions and classes developed for the application of inverse methods to the optimization of carbon dioxide surface fluxes using atmospheric mixing ratio measurements. Derivations of the equations and terminology are given in the section on the theory. The User Guide walks through an example of using the code.
